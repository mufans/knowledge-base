# 个人技术知识库

> AI Agent · 移动端开发 · 开源项目分析

## 关于

这个知识库记录了我在AI Agent、移动端开发、开源项目分析等方面的技术积累。通过每日自动化采集→分析→提炼的流程，持续构建结构化的个人知识图谱。

## 知识库结构

- **原始采集**（raw/inbox）— 每日自动采集的技术新闻、论文、GitHub项目、技术动态
- **知识提炼**（wiki/）— 从原始数据中提炼的深度分析页面
  - 概念：抽象的技术思想和方法论
  - 实体：具体的项目、工具、产品
  - 来源摘要：文章和论文的详细摘要
  - 综合分析：跨源对比分析

## 质量标准

每篇知识页面遵循统一规范：
- 包含标签、评分、出处链接
- 避免泛泛而谈，体现具体技术细节
- 与其他知识页面建立交叉引用

## 更新频率

- 每日自动采集 5-6 个信息源
- 每日自动提炼高价值内容
- 知识页面持续迭代更新
