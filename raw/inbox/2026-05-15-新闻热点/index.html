    🤖 AI / 大模型¶

    1. Redis 之父发布 DeepSeek V4 Flash 专用推理引擎 ds4 Redis 作者 antirez 为 DeepSeek V4 Flash 打造专用推理引擎，极致压榨性能。

    2. 腾讯把AI塞入一切 腾讯全面将AI能力整合到各产品线，从社交到工具全覆盖。

    3. 终端 AI 编程助手 Reasonix 开源：长会话成本暴降 80% Reasonix 开源，针对 DeepSeek 缓存机制深度优化，大幅降低长对话成本。

    4. DeepSeek 大范围开放识图模式：不是 OCR，是真看图说话 DeepSeek 视觉理解能力升级，支持真正的图像语义理解而非简单文字识别。

    5. curl 之父实测 Anthropic Mythos 模型，扫描 17.6 万 C 代码仅发现 1 个低危漏洞 curl 作者 Daniel Stenberg 用 Anthropic 新模型对自己的 C 项目做安全审计，效果令人失望。

    6. AI 工具作弊流行迫使普林斯顿推翻无人监考制度 AI 作弊泛滥导致普林斯顿大学不得不恢复人工监考。

    7. Meta 员工抗议公司对其鼠标移动和按键的跟踪 Meta 用 AI 监控员工工作行为引发内部强烈反弹。

    8. AI 员工支持多会话并行处理：NocoBase 开源低代码平台 NocoBase 更新支持 AI Agent 多会话并行，提升无代码平台智能化水平。

    💻 编程 / 开发工具¶

    1. Mojo 编程语言发布 1.0.0b1：像 Python 一样写，像 C++ 一样跑 Modular 公司的 Mojo 语言终于接近 1.0，承诺 Python 语法 + C++ 性能。

    2. Ant Design 6.4.0 发布 蚂蚁企业级 React UI 库发布新版本。

    3. Gradle 9.5.1 发布 构建工具 Gradle 发布新版。

    4. Semi Design v2.97.0 发布，抖音企业级 UI 库 字节跳动旗下 Semi Design UI 库更新。

    5. nginx 1.30.1 发布 Web 服务器/反向代理 nginx 发布新版。

    6. PHP 酝酿二十年来最大许可证变革：全面切换至 BSD License PHP 计划从自定义许可证迁移到标准 3-clause BSD，影响整个 LAMP 生态。

    7. 停止 Vibe Coding 七个月后，一位开发者决定手写全部代码 反思 AI 辅助编程的利弊，有人选择回归手写代码。
