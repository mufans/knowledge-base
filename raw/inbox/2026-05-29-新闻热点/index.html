    🤖 AI/大模型¶

    1. Anthropic融资650亿美元，估值达9650亿美元超越OpenAI Anthropic完成新一轮融资，投后估值9650亿美元首次超过OpenAI，由Altimeter Capital、Dragoneer等领投，年化收入突破470亿美元。

    2. 面壁智能发布全球首个完全AI编写的训练框架，速度反超英伟达 面壁智能用AI重写国产算力软件，训练速度超越英伟达原生方案，目标是把整个训练链路用AI重写一遍。

    3. 微软拟下周Build大会发布一系列全新AI模型 微软将在Build开发者大会发布包括编码模型在内的多款AI模型，提升GitHub Copilot竞争力，涉及转录、推理、语音和图像处理。

    4. DeepSeek从零造中国版Claude Code，招人组队 DeepSeek宣布启动独立项目，从零打造中国版Claude Code，模型之外全部自建（Harness框架），正在大规模招聘。

    5. Claude Code首发「自愈」功能，一锤砸碎开发者6大噩梦 Claude Code推出自动修复功能，能自动检测和修复代码中的常见问题，大幅提升AI辅助编程的可靠性。

    🛠️ 编程/开发工具¶

    1. Zod作者推出Pullfrog：跑在GitHub Actions里的开源AI编程Agent Pullfrog是一个开源的CodeRabbit替代方案，完全运行在GitHub Actions中，为PR提供AI代码审查。

    2. C++之父开撕AI Coding：资深开发者宁愿退休也不愿伺候AI生成的代码 Bjarne Stroustrup公开批评AI生成代码的质量问题，认为资深开发者不应被强迫维护AI生成的低质量代码。

    3. Node.js拟内置虚拟文件系统，AI生成代码引争议 Matteo Collina提议为Node.js核心添加node:vfs模块，因大量使用AI开发而引发代码审核与功能必要性的社区争议。

    4. Agentic Coding + ClickHouse: 1人1栈1应用，AI全栈几天搞定 展示如何利用AI智能体编程+ClickHouse，一个人用一套技术栈在几天内构建完整数据应用。

    📱 移动端¶

    1. 苹果Siri迎来近十五年最大改版，WWDC将公布iOS 27更新 苹果将在6月8日WWDC上发布重新设计的Siri界面和全新聊天机器人应用，这是Siri自2011年推出以来的最大改版。

    2. YouTube将自动标记AI生成视频 YouTube宣布自动检测并标记AI生成视频，长视频在播放器下方显示标签，短视频以叠加层形式展示，提升内容透明度。

    3. 小米模型永久降价99%，雷军还要再砸600亿 小米宣布永久性翻新整个模型定价体系，降幅达99%，即使利润腰斩也要在AI领域持续投入600亿。

    4. Gemma 4多词元预测：生成速度最高提升约3倍 Google Gemma 4结合MTP草稿模型借助投机解码，推理提速约3倍且不影响输出质量，对端侧部署意义重大。

    ☁️ 云原生/后端¶

    1. 腾讯云DatabaseClaw：让AI Agent真正接管生产数据库 腾讯云发布DatabaseClaw平台，以四重安全纵深和Skill生态，让AI Agent从"能用"走向"可托付"地管理生产数据库。

    2. Azure Logic Apps为智能体工作流新增沙箱化代码解释器 微软为Azure Logic Apps推出Hyper-V隔离沙箱代码解释器，AI智能体可安全生成并运行多种编程语言代码。

    3. 昇腾管推理、鲲鹏管Agent，Agentic AI让CPU重回舞台中央 华为提出昇腾芯片负责推理、鲲鹏CPU负责Agent的新架构，在Agentic AI时代重新定义CPU的角色。

    4. Google Cloud在BigQuery中引入跨引擎Apache Iceberg支持 Google宣布BigQuery支持无服务器Iceberg REST Catalog，可在Spark、Flink、Trino等引擎间共享Iceberg表。

    2. 华为"韬定律"5年内冲刺等效1.4nm制程，麒麟昇腾将先后量产 中国首次提出半导体演进新原则——逻辑折叠技术，将先后在麒麟秋季新品和昇腾990芯片量产落地。
